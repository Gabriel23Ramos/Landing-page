#  English Teacher Landing Page

Uma landing page moderna e altamente otimizada para captação de alunos de inglês, com foco em conversão, experiência do usuário e design premium.

# Sobre o projeto

Este projeto é uma landing page desenvolvida para promover aulas particulares de inglês com foco em:

- Conversão de leads
- Experiência do usuário (UX)
- Design moderno e responsivo
- Interatividade (quiz de nível)

A aplicação apresenta um fluxo simples e eficiente:
1. Usuário acessa a página
2. Realiza um quiz interativo
3. Recebe seu nível de inglês
4. É direcionado para contato via WhatsApp

# Funcionalidades

-  Landing page focada em conversão
-  Quiz interativo de nível de inglês
-  Sistema de pontuação e classificação (Beginner, Intermediate, Advanced)
-  Design totalmente responsivo
-  Animações suaves (scroll reveal, hover effects)
-  Integração direta com WhatsApp
-  Navegação fluida (scroll suave)
-  UI moderna com efeito glassmorphism e gradientes

# Tecnologias utilizadas

- HTML5
- CSS3 (com design system customizado)
- JavaScript
- Google Fonts
